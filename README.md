# Solfeggio

A web app for exploring solfeggio frequencies, visualizations, and interactive controls.

## Features
- Frequency dial and display
- Audio playback of solfeggio tones
- Mandala and particle visualizations
- Responsive UI

## Getting Started
1. Clone the repository:
   ```
   git clone https://github.com/t3kama07/solfeggio.git
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the app:
   ```
   npm start
   ```
